# 📊 Customer Shopping Behavior Analysis

## 🚀 Overview

This project is a complete end-to-end data analytics workflow focused on analyzing customer shopping behavior. It involves loading data in Python, performing data cleaning and exploratory data analysis (EDA), running SQL queries, and building a dashboard for business insights.

---

## 📂 Dataset

* Contains ~3,900 customer transactions
* Includes demographics, purchase behavior, product details, and ratings
* Some missing values handled during preprocessing

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy, Matplotlib)
* SQL (PostgreSQL / MySQL / SQL Server)
* SQLAlchemy
* Power BI (for dashboard)
* Jupyter Notebook

---

## ⚙️ Steps Performed

1. **Data Loading** – Imported dataset using Pandas
2. **Data Cleaning** – Handled missing values and corrected data types
3. **EDA** – Explored patterns using statistics and visualizations
4. **Feature Engineering** – Created new features like age groups and segments
5. **SQL Integration** – Loaded data into database and ran queries
6. **Dashboard Creation** – Built Power BI dashboard for insights
7. **Reporting** – Summarized findings in PPT/report format

---

## 📊 Dashboard & Results

* Identified high-value customer segments
* Analyzed revenue trends across gender and shipping types
* Found subscription users contribute higher revenue
* Observed that express shipping users spend more
* Highlighted top-rated and high-performing products

---

## ▶️ How to Run

1. Clone the repository
2. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib sqlalchemy psycopg2
   ```
3. Open Jupyter Notebook
4. Run the notebook step-by-step
5. (Optional) Connect to database using your credentials
6. Open Power BI file to view dashboard
